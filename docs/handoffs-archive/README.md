# Handoffs Archive

Este diretório guarda handoffs encerrados (feature entregue ou sem atividade por 90 dias).

## Regras de arquivamento
- Mover a pasta inteira da feature: `docs/handoffs/<feature>/` → `docs/handoffs-archive/<feature>/`.
- Manter a estrutura e o versionamento originais.
- Não reabrir handoffs arquivados; crie uma nova pasta em `docs/handoffs/` se a feature voltar.
