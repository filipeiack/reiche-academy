# Handoffs — comunicação entre agentes

Há duas bases principais para navegar pelos handoffs:

1. **Pastas por feature/assunto** (`/docs/handoffs/<feature>/`).
2. **Guias centrais** (`docs/governance.md` e `docs/handoffs/guidelines.md`).

Use o README aqui para lembrar:

- O fluxo oficial: `business-v1.md → dev-vN.md → qa-vN.md`.
- A versão avança quando o agente anterior indica “NÃO CONFORME”.
- QA só atua quando o Dev Agent sinaliza “Pronto para QA”.
- Após 90 dias sem atividade ou após merge, mova a pasta para `docs/handoffs-archive/`.

Para detalhes de escrita, nomenclatura e arquivamento, leia `docs/handoffs/guidelines.md`.
