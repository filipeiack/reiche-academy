# Testes E2E - Playwright

## Estrutura de Testes

Este diretório contém os testes end-to-end (E2E) usando Playwright.

### Fixtures

Coloque arquivos de teste (imagens, documentos, etc.) nesta pasta para usar nos testes.

Por exemplo:
- `test-avatar.png` - Imagem de teste para upload de avatar
- `test-document.pdf` - Documento de teste
